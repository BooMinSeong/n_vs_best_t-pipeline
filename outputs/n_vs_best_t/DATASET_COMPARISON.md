# Dataset-level Performance Comparison

N={32, 64, 128, 256, 1024, 2048} 기준, stratum=overall, 각 전략별 accuracy 비교.

전략 설명:
- **T=0.1 / T=1.0**: 고정 temperature로 N개 샘플 생성 후 majority vote
- **Random T**: 12개 T에서 균등 샘플링 후 majority vote
- **\MethodEqualMix{}**: 12개 T에서 동일 비율 샘플링 후 majority vote
- **\MethodCV{} (Consensus Vote)**: temperature별 majority → 최종 majority
- **Best T (oracle)**: 최적 T를 사전에 알고 있는 upper bound
