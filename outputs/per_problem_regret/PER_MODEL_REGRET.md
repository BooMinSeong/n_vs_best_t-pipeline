# Per-Model Mean Regret by Budget $N$ ($R_1$ and $R_2$)

Mean per-problem regret, **averaged per model over the datasets that model covers**, as a
function of the sampling budget $N$. Two regret kinds (see `SYNTHESIS_R1.md` §0–1):

- $R_1 = a^\*(p)-a(p,s)$ — vs. the per-problem oracle (unattainable ceiling).
- $R_2 = a(p,T^\dagger)-a(p,s)$ — vs. the best fixed temperature (operational regret).

Each cell is the **unweighted mean over that model's datasets** within the grouping (every
dataset counts once, regardless of problem count). Three groupings: **ALL** (every dataset),
**MATH/GSM8K** (`gsm8kfull, math1k, math500, mathfull`), **AIME** (`aime2023–2026`).

### Dataset coverage per model

| model | MATH/GSM8K | AIME |
|---|---|---|
| Llama-3.2-3B | gsm8kfull, mathfull | — |
| Phi-4-mini-instruct | gsm8kfull, math500 | aime2023–2026 |
| Qwen2.5-3B | gsm8kfull, math1k, mathfull | aime2023–2026 |
| Qwen3-4B-Instruct-2507 | — | aime2023–2026 |

> Coverage is uneven, so **ALL** is not comparable across models (Llama's ALL = MATH/GSM8K
> only; Qwen3-4B's ALL = AIME only). For cross-model comparison use the matched grouping
> (MATH/GSM8K or AIME), not ALL.

Strategies: Fixed $T{=}0.1$, Fixed $T{=}1.0$, Random-$T$, Temperature Pool (`equal_mix`),
Temperature Consensus (`consensus_vote`).

## Notes

- **$R_1 = R_2 + \Delta_{\text{oracle}}$**, a strategy-independent per-model/per-group/per-$N$
  shift. So within any one sub-table the strategy ordering and gaps are identical between
  $R_1$ and $R_2$; only the absolute level differs (large on MATH/GSM8K, small on AIME where
  the oracle gap is near zero). See `SYNTHESIS_R1.md` §0 — under the unknown-problem
  assumption $R_2$ is the operational regret.
- **Source.** `summary_distribution_stats.csv` (`regret_mean`), unweighted mean over each
  model's datasets in the grouping. Numbers carry the ~1 pp upward oracle bias in $R_1$.
